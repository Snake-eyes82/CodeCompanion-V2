# Introduction to Python: Hello, World!
**Main Concept:** Understanding the basic structure of a Python program and using the print() function to display output.
## What is Python?
Python is a versatile and widely-used programming language known for its readability and ease of use. It's used in various domains, from web development to data science and machine learning.  We'll start with the simplest program possible!
## The 'Hello, World!' Program
The traditional first program in any language is to print 'Hello, World!' to the console.  In Python, this is incredibly simple.
```python
print('Hello, World!')
```
## Understanding the Code
The `print()` function is a built-in function in Python. It takes an argument (in this case, the string 'Hello, World!') and displays it on the console.  The string is enclosed in single quotes (' ') or double quotes (" "). Both work the same way.
## Running Your First Program
To run this code, you'll need a Python interpreter.  You can either use an online Python interpreter or install Python on your computer.  Once you have Python set up, save your code in a file (e.g., hello.py) and run it from your terminal using `python hello.py`.
## Adding Comments
Comments in Python start with a # symbol. They are ignored by the interpreter but are essential for making your code understandable. Comments help you and others understand what your code does.
```python
# This is a comment
print('Hello, World!')
```
