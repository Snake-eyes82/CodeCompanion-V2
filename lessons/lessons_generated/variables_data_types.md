# Python Variables and Data Types
**Main Concept:** Understanding variables as containers for data and the different types of data Python can handle.
## What are Variables?
In programming, a variable is like a labeled container that holds information.  You give it a name, and you can store different types of data inside it.  Think of it like a box where you can put things – numbers, words, or even more complex information.
## Data Types
Python has several built-in data types. Some of the most common are:

* **Integers (int):** Whole numbers (e.g., 10, -5, 0)
* **Floating-point numbers (float):** Numbers with decimal points (e.g., 3.14, -2.5, 0.0)
* **Strings (str):** Sequences of characters (text), enclosed in single (' ') or double (" ") quotes (e.g., 'Hello', "Python")
* **Booleans (bool):** Represent truth values, either True or False
## Declaring Variables
In Python, you don't explicitly declare the type of a variable.  Python infers the type based on the value you assign. You simply give it a name and assign a value using the equals sign (=).
## Examples
```python
age = 30
name = "Alice"
height = 5.8
is_student = True
print(age, name, height, is_student)
```
## Variable Naming Conventions
Choose descriptive names for your variables.  Use lowercase letters, and separate words with underscores (snake_case). Avoid using reserved keywords (like 'if', 'for', 'while').
